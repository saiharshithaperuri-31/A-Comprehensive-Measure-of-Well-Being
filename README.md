# A Comprehensive Measure of Well-Being

## 📌 Project Overview
The **A Comprehensive Measure of Well-Being** project is a Machine Learning-based web application that predicts the Human Development Index (HDI) category using key socio-economic indicators. The application analyzes factors such as life expectancy, education, and Gross National Income (GNI) to provide insights into a country's level of human development.

---

## 🎯 Objectives
- Analyze Human Development Index (HDI) data.
- Build a Machine Learning model for HDI prediction.
- Develop a user-friendly web application using Flask.
- Provide meaningful insights based on socio-economic indicators.

---

## ✨ Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Machine Learning model training
- HDI prediction through a Flask web application
- Simple and responsive user interface
- Model serialization using Pickle

## ⚙️ Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project folder

```bash
cd A-Comprehensive-Measure-of-Well-Being
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the Flask application

```bash
python app.py
```

5. Open your browser and visit

```
http://127.0.0.1:5000/
```

## 📊 Dataset

The project uses a Human Development Index (HDI) dataset containing indicators such as:

- Human Development Index (HDI)
- Life Expectancy at Birth
- Expected Years of Schooling
- Mean Years of Schooling
- Gross National Income (GNI) per Capita
